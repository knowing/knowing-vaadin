This extension provides Vaadin UI components to work with Knowing.

Requirements:

* [Vaadin OSGi Addon](http://vaadin.com/directory#addon/vaadin-osgi)
* Jetty
* Eclipse Helios 3.6.2
* slf4j 1.6.1 / logback
